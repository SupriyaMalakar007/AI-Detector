# AI-Detector
An AI-powered text detection system that analyzes and classifies content as human-written or AI-generated using Machine Learning and NLP techniques.
📌 Project Overview

AI-Detector is a Machine Learning and NLP-based project that detects whether a given text is written by a human or generated by Artificial Intelligence. The system analyzes text patterns, sentence structure, and writing style to classify the content.

🚀 Features

Detect AI-generated text
Human vs AI text classification
Simple and user-friendly interface
Fast text analysis
Supports custom text input
PDF/Image text extraction support (if added)

Human-written text samples

The model is trained using labeled text data for classification.
